# PREGÃO Marketplace

Plataforma de comércio e serviços para empresas em Angola.

## Funcionalidades

- ✅ Marketplace completo para produtos e serviços
- ✅ Gestão robusta de stocks (FIFO/LIFO)
- ✅ Sistema de pedidos e carrinho
- ✅ Módulo de entregas com motoristas
- ✅ Backoffice administrativo
- ✅ API RESTful completa
- ✅ Autenticação Sanctum
- ✅ Testes automatizados

## Tecnologias

- Laravel 10
- MySQL 8
- Redis
- Docker & Sail
